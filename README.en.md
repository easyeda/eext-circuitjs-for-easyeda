# CircuitJS1 Simulator

A locally embedded CircuitJS1 circuit simulator for EasyEDA Pro.

## Features

### ✅Embedded CircuitJS1 Simulator

Run CircuitJS1 simulation locally inside EasyEDA, no internet required.

![alt text](images/gif1.gif)

### ✅One-Click Export to EasyEDA

**BETA: Complex simulations are not yet supported.**

Click "Export to LCEDA" in the CircuitJS1 menu bar to automatically place components and wires on the schematic.

![alt text](images/gif2.gif)

- Auto compensation wires for component pins
  - VCC and GND net flags created automatically
